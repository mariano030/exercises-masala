1fr stands for fractional unit.

grid-template-*
can take px % and fr as measurements.

grid-column: 1 / 4; ist the same as these two

grid-column-start: 1; grid-column-end: 4;

With

grid-column: 1 / -1;
you are using -1 to address the last column in the row.

You can use gird-template-columns: repeat(4, 1f); instead of gird-template-columns: 1fr 1fr 1fr 1fr;

grid-auto-flow: row dense
This will make the implicit elements fill the empty spots!
